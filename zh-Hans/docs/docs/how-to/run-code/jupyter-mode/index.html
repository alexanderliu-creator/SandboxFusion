Jupyter Mode

除了把代码作为一整个脚本执行，沙盒还支持通过Jupyter运行一系列的代码片段。 Jupyter模式的主要区别在于：

  • 在一个Cell因为异常而中止运行时，后续的Cell仍然会继续执行
  • 在标准的stdout和stderr输出以外，新增了两个输出流：
    • display流，支持富文本，每个Cell最后一条语句的结果、matplotlib绘制的图表都会在这里输出
    • error流，未被捕获的异常会在这里输出

下面的Python脚本展示了Jupyter模式的使用方式和特性：

import json
import base64
import requests

cells = [
    '''
a = 123
# display stream
a
    ''',
    '''
# stdout & stderr stream
print(a)
import sys
sys.stderr.write(str(a))
    ''',
    '''
raise Exception("This exception doesn't interrupt cell execution")
    ''',
    '''
import matplotlib.pyplot as plt

plt.plot([1, 2, 3], [1, 2, 3])
    '''
]

response = requests.post('http://localhost:8080/run_jupyter', json={
    'cells': cells,
    'kernel': 'python3'
})

print(json.dumps(response.json(), indent=2))
