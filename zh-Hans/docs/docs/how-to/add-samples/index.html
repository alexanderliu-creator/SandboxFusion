Using Datasets 部分描述了如何使用某个数据集，这里介绍如何用自定义数据在某个数据集上运行。

对于沙盒实现的外部数据集，直接参考样例中输入给沙盒的数据格式即可，各种数据字段的含义参考对应的数据集。

沙盒有两个原创的格式，基于测试代码的 AutoEval 数据集和基于标准输入输出的 CommonOJ 数据集，足以满足绝大部分评估场景。 其数据格式请参考对应的链接。

下面是一个通过请求参数传入数据的脚本，起作用的参数为 config 中的

  • dataset_type: 数据集类型
  • provided_data:
    • 本次请求要使用的数据。 如果该参数存在，id参数无效，会以传入数据为准。
    • 数据结构为一个 json object 。
提示

部分现有数据集的实现中，会依赖请求中的 dataset 参数根据数据集名字的不同执行特定逻辑，比如 humaneval 数据集，是根据数据集名字判断使用的语言的，所以传入的数据最好保持 dataset 参数不变。

如果 dataset_type 对应的实现并没有针对 dataset 参数做特殊处理，就无所谓了。
